Co dostanete?

Odkaz, kde si můžete nakódovanou šablonu prohlédnout. V ceně je možné 5 revizí. Jakmile nám práci odsouhlasíte (do 7 dnů, pak je automaticky za odsouhlašenou a vystavujeme fakturu), pošleme Vám odkaz ke stažení balíku v .zipu a fakturu se 14denní splatností.

Garance

Garantujeme dodání výsledného balíku do 3 pracovních dnů. Pokud termín nestihneme lorem ipsum dolor sit amet consectetur adipiscing. Výjimkou jsou objektivní důvody. Nebo se dohodneme jinak.

Technicky

HTML, CSS i JavaScript posíláme vždy v minifikované podobě, ale v balíku najdete i zdrojové kódy s komentářem.

  • mobilní verze
  • HTML5, CSS3
  • umíme jQuery
  • používáme SASS
  • Retina Ready
  • tiskový styl
  • měřící kódy
  • sémantický, minimalistický a čistý kód
  • validní kód a rychlý kód
  • responzivní design

Co potřebujeme

  • vrstvené a dobře popsané PSD
  • použité fonty vč. webové verze (pozor na diakritiku)
  • favicon
  • komentář grafika
  • odkaz na podklady (např. set obrázků pro slider)
  • responzivní design ano/ne
  • Retina Ready ano/ne (vyžaduje obrázky ve dvojnásobném rozlišení)
  • termín odevzdání
  • pozice elementů Hx
  • uveďte konkrétní slider, pokud chcete
  • používáte-li ikonky, dejte nám odkaz na icon set (nebo pošlete je-li placený)
  • pořadí obsahových bloků pro SEO
  • texty pro nadpis stránek (title) a meta tag description
  • můžeme si dát do referencí?
